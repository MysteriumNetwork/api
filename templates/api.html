Mysterium.Network API
Version: 1.0
URL Base: https://mvp.mysterium.network:5000
Params should be encoded in JSON and passed as HTTP body data.
Returned values are encoded in JSON. API reqest returns status code. For example status code 200 means success.

POST /v1/node_register
Params: Proposal
Proposal Example: "service_proposal": { "id": 1, "format": "service-proposal/v1", "provider_id": "node1" }
Returns:

GET /v1/proposals
Params: node_key (optional)
Returns: proposals

POST /v1/client_create_session
Params: node_key, session_key (optional)
Returns: session_key, connection_config

POST /v1/node_send_stats
Params: node_key, sessions: [session_key, bytes_sent, bytes_received]
Returns: sessions: [session_key, is_session_valid]

POST /v1/client_send_stats
Params: session_key, bytes_sent, bytes_received
Returns: session_key, is_session_valid
